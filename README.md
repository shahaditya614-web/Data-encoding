# 🔤 Data Encoding using Python

A beginner-friendly implementation of **Data Encoding** techniques using **Python**, **Pandas**, and **Scikit-learn**. This project demonstrates how to convert categorical data into numerical format using popular encoding methods, making datasets suitable for Machine Learning algorithms.

---

# 📌 Project Overview

This project demonstrates different **Data Encoding** techniques used in data preprocessing. It includes methods such as **Label Encoding**, **One-Hot Encoding**, and **Binary Encoding** to transform categorical variables into numerical values.

The notebook is designed for **Google Colab** and is suitable for beginners learning **Machine Learning** and **Data Preprocessing**.

